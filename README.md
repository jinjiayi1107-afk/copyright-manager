# 版权管理系统

一个基于 Flask + Vue.js 的出版社版权管理系统。

## 功能模块

### 核心模块
1. **数据看板** - 展示各类统计数据和图表
2. **外版图书档案** - 图书基本信息管理、状态流转
3. **合同管理** - 合同存档、版税信息、到期提醒
4. **外商库** - 国外出版社信息管理
5. **译者库** - 译者信息管理
6. **意向选题库** - 选题意向管理
7. **版税管理** - 版税支付记录

### 已实现功能
- ✅ 所有模块的增删改查 (CRUD)
- ✅ 状态管理和流转
- ✅ 数据统计看板
- ✅ 图表可视化 (饼图)
- ✅ 搜索筛选功能
- ✅ 表单验证
- ✅ 响应式布局

## 技术栈

- **前端**: HTML5 + CSS3 + JavaScript + Vue.js 3
- **后端**: Python Flask
- **数据库**: SQLite
- **图表**: ECharts 5

## 项目结构

```
copyright_manager/
├── app.py              # Flask 后端主文件
├── database.py          # 数据库操作模块
├── requirements.txt     # Python 依赖
├── static/
│   ├── index.html      # 主页面
│   ├── css/
│   │   └── style.css   # 样式文件
│   └── js/
│       └── app.js      # 前端逻辑
└── uploads/            # 上传文件目录 (自动创建)
```

## 快速启动

### 1. 安装依赖

```bash
cd 版权管理系统/code
pip install -r requirements.txt
```

### 2. 启动服务

```bash
python app.py
```

### 3. 访问系统

打开浏览器访问: **http://localhost:5000**

## 数据库

系统使用 SQLite 数据库，数据库文件 `copyright_manager.db` 会在首次运行时自动创建。

## 枚举值说明

### 意向选题状态
- 待洽谈
- 洽谈中
- 已签约
- 已放弃

### 图书状态
- 意向阶段
- 合同签约阶段
- 翻译中
- 编辑中
- 已出版
- 近期需续约
- 已过期
- 已废弃

### 合同状态
- 草稿
- 已签约
- 执行中
- 已到期
- 已作废

### 版税类型
- 统一版税率
- 阶梯版税率

### 有效期计算方式
- 签订日期开始
- 出版日期开始
